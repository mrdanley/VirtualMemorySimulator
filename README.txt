How to compile and run file:

main directory: VirtualMemorySimulator

1) put test file into VirtualMemorySimulator\test_files
2) cd into VirtualMemorySimulator\src
3) compile all classes by using 'javac *.java'
4) run program by using 'java Main' with test filename afterwards
5) test results file is in VirtualMemorySimulator\test_results





example run:

<current location VirtualMemorySimulator>

COMMAND LINE:
mv test_file_1.txt test_files\test_file_1.txt
cd src
javac *.java
java Main test_file_1.txt

<test_file_1_results.csv now located in VirtualMemorySimulator\test_results>
